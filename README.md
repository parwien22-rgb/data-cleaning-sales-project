# Data Cleaning Sales Project

## Overview
This project demonstrates data cleaning and basic analysis using Python and Pandas. I transformed a messy, real-world sales dataset into a clean, structured format ready for reporting or dashboards.

## Why I built this project
Coming from a background in healthcare and hospitality, I am transitioning into the ICT and data field. This was one of my first hands-on projects to build practical data skills and understand how to work with real, imperfect data.

## What I did
- Loaded and explored raw sales data
- Identified and handled missing values and duplicates
- Standardized column names and data structure
- Performed basic sales analysis (total and average)
- Exported a cleaned CSV dataset

## What I learned
- How critical proper data cleaning is (“garbage in = garbage out”)
- Efficient use of Pandas for data inspection, cleaning and transformation
- The importance of reproducible code and version control with Git

## Tools Used
- Python
- Pandas
- Git & GitHub

## Result
A clean, analysis-ready dataset that can be used for reporting, visualizations or further analysis.
